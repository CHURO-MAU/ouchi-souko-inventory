# 📦 日用品在庫管理アプリ

シンプルで使いやすい日用品の在庫管理Webアプリケーションです。

## 特徴

- ✨ シンプルで直感的なUI
- 💾 ブラウザのLocalStorageを使用したデータ永続化
- 📊 在庫状態の自動判定（在庫あり/在庫不足/在庫切れ）
- 🔍 カテゴリ別フィルタリング
- ⚠️ 在庫不足アラート表示
- 📱 レスポンシブデザイン対応

## 使い方

1. ブラウザで `index.html` を開きます
2. 「新しい日用品を追加」フォームから商品を登録します
   - **商品名**: 管理したい日用品の名前（例：トイレットペーパー）
   - **在庫数**: 現在の在庫数
   - **最低在庫数**: この数以下になると「在庫不足」として警告されます
   - **カテゴリ**: 商品のカテゴリを選択
3. 登録した商品は下部の一覧に表示されます
4. 「＋」「−」ボタンで在庫数を増減できます
5. 「削除」ボタンで商品を削除できます

## 機能

### 在庫管理
- 商品の追加・削除
- 在庫数の増減（＋/−ボタン）
- 在庫状態の自動判定
  - **在庫あり**: 在庫数が最低在庫数を上回っている
  - **在庫不足**: 在庫数が最低在庫数以下
  - **在庫切れ**: 在庫数が0

### フィルタリング
- カテゴリ別表示（日用品、食品、洗剤、衛生用品、その他）
- 在庫不足のみ表示

### データ保存
- ブラウザのLocalStorageを使用
- ページを閉じてもデータは保持されます
- ブラウザのキャッシュをクリアすると削除されます
